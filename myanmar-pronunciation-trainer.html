ミャンマー語 発音練習

お手本フレーズを聞き、自分の発音を録音して、認識結果との一致度・声の高低の変化を確認できます。

1お手本フレーズ

「Google翻訳の音声で再生」は登録不要ですぐ使えます（非公式のため将来使えなくなる可能性があります）。
このフレーズのお手本リズムはまだ記録されていません（下のボタンで記録できます）。

上の「Google翻訳の音声で再生」で十分な場合は、以下は不要です。他の外部サイトで生成した音声ファイルをお持ちの場合のみ、こちらに保存してご利用ください。

このフレーズの保存済み音声：なし

2自分の発音を録音

00:00
マイクの使用許可が必要です。「録音開始」を押すと許可を求めます。
録音が終わると、お手本の直後に自分の発音を自動再生して聞き比べられます。

3客観比較（お手本との声の高低パターン）

–
お手本と自分の発音の「声の高低（トーン）パターン」の近さ。上の「🎯 お手本の声をマイクで記録して比較用に保存」を一度行うと、以降この録音のたびに自動採点されます。
録音すると、声の高さ（基本周波数）の推移を表示します。（グラフ：■お手本 / ■あなた）
お手本のリズムを記録すると、ここに比較結果が表示されます。
このアプリの限界について： 「Google翻訳の音声で再生」は非公式の方法のため、将来Google側の仕様変更で使えなくなる可能性があります。 「声の高低パターンの近さ」は、Googleの音声をスピーカーで再生しマイクで録音して比較する仕組みのため、周囲の騒音やスピーカー音量の影響を受けやすく、あくまで参考値です。声調（トーン）そのものの絶対的な正誤を機械的に判定する技術ではありません。 以前あった、録音した声をブラウザ内蔵の音声認識でテキスト化する自動採点機能は、ミャンマー語への対応が不安定だったため廃止しました。 発音の正確さの最終確認は、可能な限りミャンマー語ネイティブスピーカーへの確認も併用してください。

Web Speech API / Web Audio API を使用（ブラウザ内で完結。録音データは外部送信されません）
